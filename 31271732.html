Importancia de Git

Git es importante porque ayuda a mantener un control de versiones para asi si hay algun cambio realizado que el sistema colapsa o tiene algun problema de compilacion puede regresar a cuando si funcionaba para asi rectificar que paso y el porque. tambien ayuda a mantener un control por ramas que ayuda a que las modificaciones cuando son echas por un equipo de desarrolladores no afecten la rama principal y asi ayuda a que no acurran fallos inesperados.
