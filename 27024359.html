El control de versiones es importante para que un equipo que trabaja en conjunto para realizar un proyecto puedan trabajar todos al mismo tiempo sin tener que esperar a que los demas hagan su parte para poder ellos hacer la suya, y ademas permite que todos tengan acceso a todos los datos al mismo tiempo y que se lleve un registro de quien realiza cada cosa. Lo que mas me gusto de la unidad git es que nos enseña los comandos y que hace cadad uno, la importancia de usar el git para trabajar en proyectos y el hecho de que esta herramienta nos ayuda a llevar un registro y control de las versiones que se van realizando y en caso de fallo poder dar mas facilmente con el error que se cometio. o si varios modificaron el mismo archivo poder ver las modificaciones y corregir el problema.
